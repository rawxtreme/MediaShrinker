# MediaShrinker

MediaShrinker is a modern Android utility app that compresses images to exact KB sizes quickly and easily.

## Features

- Compress images to custom KB size
- Save compressed images directly to Gallery
- Share compressed images instantly
- Modern dark UI
- Fast image processing
- About App Section
- Android 11+ support

## Folder Location

Compressed images are saved in:

DCIM/MediaShrinker

## Built With

- Kotlin
- Android SDK
- AndroidIDE
