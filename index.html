MediaShrinker

Powerful all-in-one image toolkit for Android. Built entirely on a phone using AndroidIDE.

View on GitHub

About MediaShrinker

MediaShrinker is designed to simplify your everyday image tasks. Compress photos, create PDFs, resize images, convert formats, remove metadata, and manage your files efficiently without subscriptions, advertisements, or hidden charges.

Features

  • Smart Image Compression
  • Batch Compression
  • PDF Creation
  • Image Resizer
  • Format Converter
  • Metadata Remover
  • Image Comparison Tool
  • Compression History
  • Storage Analytics Dashboard
  • Image Information Viewer
  • Rename Before Saving
  • Instant Sharing
  • Custom Save Locations
  • Animated Splash Screen
  • Video Intro Experience

Privacy First

MediaShrinker works completely offline for all core features. Your photos never leave your device. Internet access is used only when you voluntarily check for updates through GitHub.
